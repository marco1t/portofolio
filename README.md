# Carnet de compétences PPP ING4 de Roulem Baitiche

Portfolio personnel construit avec Vite, React et TypeScript, adapté au cahier
des charges du carnet de compétences PPP ING4.

## Commandes

```bash
npm install
npm run dev
npm run build
```

## Déploiement

Le site est configuré pour GitHub Pages via `.github/workflows/deploy.yml`.
Chaque push sur `main` construit l'application et publie le dossier `dist`.
